# PVAX - *Pulseira Vibratória Auxiliadora da Experiência*

## Instalação:

1. Instale o Python 3.10 ou superior.
2. Execute o arquivo "setup.bat".
3. Aguarde a instalação das dependências.

## Execução:

1. Execute o arquivo "main.py"
   ou
2. Execute o arquivo "main.exe" (caso disponível).

## Captura de Áudio:

Se nenhum áudio for detectado, instale o VB-Cable:
https://vb-audio.com/Cable/

### Em seguida:

- Defina a saída de áudio do sistema como "CABLE Input"
- Selecione "CABLE Output" no aplicativo
